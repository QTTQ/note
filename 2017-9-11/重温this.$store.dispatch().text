this.$store.dispatch()应该放在方法里；
传入store中的actions 其他步骤跟正常的一样。
例如：
	methods: {
		ccc() {
			this.$store.dispatch("ccc")
		},
		...mapActions([
			"mbus1"
		])
	}
接收的计算属性中：
	computed: {
		...mapGetters([
			'gdata',
			'mccc'
		]),
	},    
actions中这样：
    export default {
        mbus1:({
            commit
        })=>{
            commit(types.MBUS1);
        },
        ccc:({
            commit
        })=>{
            commit(types.CCC);
        }
    }
